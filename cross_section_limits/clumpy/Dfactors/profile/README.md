This folder contains the profiles of astrophysical factors for DM decay as a function of the integration angle, generated with CLUMPY for the analyzed dSphs. It is organized into the following subdirectories:

- `Burkert` contains profiles computed adopting the Burkert profile (cored, Burkert 1995);
- `Einasto` contains profiles computed adopting the Einasto profile (cuspy, Einasto 1965).

See inside each subdirectory for the detailed description of their content.
