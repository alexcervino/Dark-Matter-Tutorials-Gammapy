This folder contains all the profiles of astrophysical factor for DM decay as a function of the integration angle, generated with CLUMPY for the analyzed dSphs according to the Einasto DM density profile (Einasto 1965).

The content of each file is organized as follows:

```
alpha_int J_dec(median) J_dec(mean) J_dec(most likely) J_dec(best chi^2) J_dec(median)_LO68% J_dec(median)_UP68% J_dec(median)_LO95% J_dec(median)_UP95%
```

with:

- `alpha_int` the integration angle;
- `J_dec(median)` the median value of astrophysical factor at that angle;
- `J_dec(mean)` the mean value of astrophysical factor at that angle;
- `J_dec(most likely)` the most likely value of astrophysical factor at that angle;
- `J_dec(best chi^2)` the value of astrophysical factor at that angle with the best $\chi^2$;
- `J_dec(median)_LO68%` the lower uncertainty on the median astrophysical factor at 68% confidence level;
- `J_dec(median)_UP68%` the upper uncertainty on the median astrophysical factor at 68% confidence level;
- `J_dec(median)_LO95%` the lower uncertainty on the median astrophysical factor at 95% confidence level;
- `J_dec(median)_UP95%` the upper uncertainty on the median astrophysical factor at 95% confidence level.
