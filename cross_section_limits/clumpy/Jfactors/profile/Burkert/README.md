This folder contains all the profiles of astrophysical factor for DM annihilation as a function of the integration angle, generated with CLUMPY for the analyzed dSphs according to the Burkert DM density profile (Burkert 1995).

The content of each file is organized as follows:

```
alpha_int J_ann(median) J_ann(mean) J_ann(most likely) J_ann(best chi^2) J_ann(median)_LO68% J_ann(median)_UP68% J_ann(median)_LO95% J_ann(median)_UP95%
```

with:

- `alpha_int` the integration angle;
- `J_ann(median)` the median value of astrophysical factor at that angle;
- `J_ann(mean)` the mean value of astrophysical factor at that angle;
- `J_ann(most likely)` the most likely value of astrophysical factor at that angle;
- `J_ann(best chi^2)` the value of astrophysical factor at that angle with the best $\chi^2$;
- `J_ann(median)_LO68%` the lower uncertainty on the median astrophysical factor at 68% confidence level;
- `J_ann(median)_UP68%` the upper uncertainty on the median astrophysical factor at 68% confidence level;
- `J_ann(median)_LO95%` the lower uncertainty on the median astrophysical factor at 95% confidence level;
- `J_ann(median)_UP95%` the upper uncertainty on the median astrophysical factor at 95% confidence level.
