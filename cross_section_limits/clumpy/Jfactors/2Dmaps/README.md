This folder contains the 2D maps of astrophysical factors for DM annihilation generated with CLUMPY for the analyzed dSphs. It is organized into the following subdirectories:

- `Burkert` contains 2D maps computed adopting the Burkert profile (cored, Burkert 1995);
- `Einasto` contains 2D maps computed adopting the Einasto profile (cuspy, Einasto 1965).

See inside each subdirectory for the detailed description of their content.
